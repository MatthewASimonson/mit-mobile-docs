1.2. Emergency¶

Commands

  • Default
  • Contacts (contacts)

1.2.1. Default¶

1.2.1.1. Description¶

This command returns an array of all of the current emergency notices for the MIT campus. Upon success, the response will have a 200 status code.

1.2.1.2. Response Schema¶

1.2.2. Contacts (contacts)¶

1.2.2.1. Description¶

This command will return a list of all the emergency contacts for MIT. Upon success, the response will have a 200 status code and will contain an array of JSON objects.

1.2.2.2. Response Schema¶

{ "type" : "array",
  "items" : {
    "type" : "object",
    "properties" : {
      "contact" : {"type" : "string", "required" : true},
      "phone" : {"type" : "string", "required" : true},
      "description" : {"type" : "string", "required" : false}
    }}}

1.2.2.3. Example Request¶

Request:

GET /api/index.php?module=emergency&command=contacts HTTP/1.0
Host: m.mit.edu
Accept: application/json
Accept-Charset: utf-8
