1.4.1. Detail¶

1.4.1.1. Description¶

This command return an object which contains detailed information for a specific event.

1.4.1.2. Parameters¶

Name Type Verb Required Notes
id integer GET Yes The id of the event to retrieve the details for.
type string GET No Defaults to ‘Events’. Other valid values can be found by calling the ExtraTopLevels command. This must match the type of the event.
