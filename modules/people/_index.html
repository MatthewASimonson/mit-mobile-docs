1.3. People¶

Commands

  • Default

1.3.1. Default¶

1.3.1.1. Description¶

This command will search the MIT directory using the string provided in the q parameter. Not including the q parameter is an error and will result in the command failing.

1.3.1.2. Parameters¶

Name Type Verb Required
q string GET Yes

1.3.1.3. Response Schema¶

{
  "type" : "array",
  "items" : {
   "type" : "object",
   "properties" : {
     "email" : { "type" : "string" },
     "givenname" : { "type" : "string" },
     "id" : { "type" : "string" },

     "name" : {
       "type" : "array",
